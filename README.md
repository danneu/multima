multima
=======

A modest Clojure MUD

Learning from [jcromartie/dunjeon](https://github.com/jcromartie/dunjeon) until I (hopefully) get on my own two feet.

## Progress

Terminal 1: Start server

    $ lein run 
    
Terminal 2: Connect client

    $ telnet localhost 5000
